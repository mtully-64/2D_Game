# The Roadmap

- Create a game loop
- Display character

- Create monsters (simple AI)
- Create battle system (hit detection)
- Create inventory
- Create merchant NPCs to buy/sell items
- Environment interaction (cutting trees)
- Title screen
- Game over screen
- Complete game
